# 温柔经期记录 - 微信小程序

一款简洁清新的女性生理周期记录小程序，全部数据本地存储，安全隐私。

## 功能特性

### 1. 基础模块
- 首次使用引导，填写个人基础数据
- 支持后期修改个人设置
- 末次经期开始日期、平均经期天数、平均周期天数可配置

### 2. 日历模块（核心）
- 月视图日历展示
- 颜色标识：
  - 经期：粉色
  - 排卵期：浅蓝色
  - 黄体期：浅黄色
  - 安全期：灰色/白色
- 用户在预测经期内手动选择「非经期」时，系统自动判定本次经期结束

### 3. 每日记录模块
- 每天一条记录
- 可选择：是否经期、心情、症状
- 历史记录查看

### 4. 周期预测
- 标准周期：28天，经期5天（可自定义）
- 自动计算下次经期、排卵期、黄体期、安全期
- 根据真实记录自动优化预测

### 5. 每日小贴士
- 根据当前阶段显示专属健康建议
- 小贴士内容可自定义修改

### 6. 数据统计
- 显示最近3个周期数据
- 周期长度、经期长度、是否规律

## 技术栈

- 框架：uni-app (Vue 3)
- 平台：微信小程序
- 存储：uni.setStorage (本地存储)

## 项目结构

```
├── pages/
│   ├── index/          # 首页日历
│   ├── records/        # 我的记录
│   ├── settings/       # 个人设置
│   └── record-detail/  # 每日记录详情
├── utils/
│   ├── storage.js      # 本地存储工具
│   └── cycle.js        # 周期计算工具
├── static/             # 静态资源
├── App.vue
├── main.js
├── pages.json
├── manifest.json
└── package.json
```

## 使用说明

1. 使用 HBuilderX 或 CLI 打开项目
2. 配置 manifest.json 中的 appid
3. 运行到微信开发者工具
4. 开始使用！

## 修改小贴士内容

编辑 `utils/cycle.js` 文件中的 `getTips` 函数即可修改各阶段的小贴士内容。

## 注意事项

- 所有数据保存在本地，卸载小程序会丢失数据
- tabBar 图标需要自行准备并放入 static/icons 目录
